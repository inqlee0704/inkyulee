# In Kyu Lee — Personal Site (GitHub Pages)

This repository contains a minimal Markdown-based site rendered by GitHub Pages (Jekyll).

## How to deploy

1. Create a repository named **USERNAME.github.io** (replace USERNAME with your GitHub username).
2. Add these files to the repository root:
   - `index.md`
   - `_config.yml`
   - `assets/InKyuLee_CV.pdf` (already included here if your CV was uploaded)
3. Commit and push to the **main** branch.
4. Visit `https://USERNAME.github.io`.

### Updating links

- Replace the placeholder `Google Scholar`, `GitHub`, and `LinkedIn` URLs in `index.md` with your real links.
